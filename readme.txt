<<起動方法>>

CD-ROMをパソコンのCD-ROMドライブに挿入すると、自動的に
ブラウザが立ち上がってメニューが表示されます。

メニューが自動的に表示されない場合は、CD-ROMにある
「index.html」をダブルクリックしてブラウザを起動してください。

メニューにある「キャラクターデッキ」および「Flash6 Player
インストーラー」は、Windowsの実行ファイルです。クリック時に
表示される「ファイルのダウンロード」ダイアログでは、
「開く(O)」を選択してください。
